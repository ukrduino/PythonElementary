NEED TO KNOW
1. What is Python and what versions we should use and why?

    Python - высокоуровневый язык программирования c динамической типизацией,
    поддержкой ООП, функционального программирования, структурного программирования и др.
    В данный момент существуют две версии Python 2 и 3.
    Для написания данного курса использована версия 2 т.к.  на большинстве
    существующих более одного года проектов используется версия 2.
    В то же время она не развивается и для новых проектов рекомендуется
    использовать версию 3.

2. How to install Python on your system.

    Нужно скачать с сайта https://www.python.org/ подходящую версию
    дистрибутива и установить (win). На unix системах Python 2 включен в
    дистрибутив. Можно также скачать и установить сборки содержащие Python
    и дополнительные пакеты например Anaconda.

3. What is a virtual environment and why do we need it?

    virtual environment - это инструмент который создаёт изолированную Python
    среду (окружение) для избегания конфликтов при необходимости использовать на одном
    компьютере несколько версий одних и тех же библиотек.
    В отдельное окружение устанавливаются отдельные версии пакетов и не
    пересекаются с другими окружениями.

4. What is pep8, and why it is recommended to follow it?

    Этот документ описывает соглашение о том, как писать код для языка Python.
    Рекомендации о стиле написания кода направлены на то, чтобы улучшить
    читаемость кода. Если код в проекте написан в едином стиле, то любой
    сможет легко его прочитать и быстро в нем разобраться.

5. How to write a simple program and run it from the command line.

    1. Создать файл с расширением .py (например Example.py) содержащий строку
      print "Hellow World"

    2. Запустить файл из директории где он расположен командой
      python Example.py (путь к python.exe должен быть в системной
      переменной PATH)
